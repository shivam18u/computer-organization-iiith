MAKING YOUR OWN INSTRUCTIONS

The following three steps will guide you in creating your own new instruciotn. Though the hardware is fixed but you can manipulate the control signals to come up with new instructions.

Step #1

You can choose from 2 types of instruction formats. You can either have an instruction with three register values or an instuction with 2 register values and one immediate value. This version supports only 1 instruction of each format simultaneously.

GETTING STARTED!!

This simulator has been designed to give a detailed and lucid explanation of the MIPS single cycle CPU architecture. It gives an insight on how an instruction is excuted in a MIPS processor. For the sake of conceptual understanding, each instruction has benn divided into 7 micro-instuctions. The simulator provides its users an option to create their own instructions using the same architecture thereby providing detailed understanding of the subject. This guide provides a brief tutorial on using this simulator.
